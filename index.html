GRIDTYPE is a tool for critiquing and archiving student's creative works.

GRIDTYPE is an assessment and archive tool to enable the documentation of the qualitative nature of design-education critique. TThis project stems from an expressed need from faculty and students, for a more definitive evaluative measure of student progress.

Student Features

  • Upload / Sort Work
  • Rate Work
  • Leave Critiques for Others
  • Add Iterations to Uploaded Works

Faculty Features

  • Create Groups / Albums
  • Rate / Comment on Work
  • Generate Department Analytics
  • Support
